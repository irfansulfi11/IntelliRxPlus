# IntelliRx+ 💊🤖

An AI-powered intelligent prescription reader and healthcare assistant that uses OCR, NLP, and machine learning to analyze medical prescriptions, detect drug interactions, and provide semantic patient history search.

---

## 🚀 Features

- 📄 Prescription image/text extraction using OCR
- 💊 Drug interaction checking
- 🧠 AI-powered semantic patient history search
- 🏥 Smart healthcare assistance
- 🔍 Medicine information retrieval
- 📊 Patient record analysis
- 🌐 Full-stack web application
- 🔐 Secure medical data handling

---

### Frontend
